Proyek ini bertujuan untuk melakukan klasifikasi gambar.

*   Dataser yang digunakan adalah Shoe vs Sandal vs Boot  Image Dataset, yang dapat diakses melalui halaman Kaggle berikut:
    https://www.kaggle.com/datasets/hasibalmuzdadid/shoe-vs-sandal-vs-boot-dataset-15k-images

*   Dataset ini memliki 15.000 gambar dan 3 kelas, yang terdiri dari 5.000 gambar Shoe, 5.000 gambar Sandal, dan 5.000 gambar Boot.
